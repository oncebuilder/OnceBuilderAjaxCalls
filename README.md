Ajax-commands-calls
===================

This is ajax commands calls controler used in oncebuilder to return ajax responses, it organize your code and let it be flexible. This is first step of publishing oncebuilder framework.

File structure
===================

```
once
config.php
once.php
class
    core.class.php
    once.core.class.php
include
    once.core.php
js
    jquery-2.1.1.js
    once.core.js
```

===================
